### Current state of Copilot

Uses the sahara-base modal (GPT 3.5-turbo) and was updated in March 2023

Has quality controls that work to prevent generating dangerous code,
like SQL injection

---

### Tips from GitHub

Write a high level description of what a file contains at the top in a comment

Follow consistent naming conventions (don't mix case)

If you want Copilot to have a certain file in context, make sure to keep that file open in a tab of your IDE
