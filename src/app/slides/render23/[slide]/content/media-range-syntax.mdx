### The Media Queries Level 4 syntax allows for more easily computing viewport ranges

<Columns count={2}>
    ```css
    /* The old way to do a mobile breakpoint */

    @media (max-width: 450px) {
        .content {
            /* Your styles */
        }
    }
    ```

    ```css
    /* The new way to do a mobile breakpoint */

    @media (width <= 450px) {
        .content {
            /* Your styles */
        }
    }
    ```

</Columns>

<Columns count={2}>
    ```css
    /* The old way to do a range */

    @media (min-width: 450px) and (max-width: 650px) {
        /* styles */
    }
    ```

    ```css
    /* The new way to do a range */

    @media (450px <= width <= 650px) {
        /* styles */
    }
    ```

</Columns>
